# JUTON POS - Advanced Point of Sale System

A premium, high-performance Point of Sale (POS) system designed for retail excellence. Built with Next.js, TypeScript, and Tailwind CSS, this system provides a seamless experience for managing inventory, transactions, and treasury operations.

## 🚀 Key Features

- **Inventory Management**: Real-time stock tracking, audit logs, and count modes.
- **Treasury Command Center**: Centralized financial management with shift tracking.
- **Mobile-First Design**: Optimized for both desktop and high-density mobile interfaces.
- **Premium UI/UX**: Professional aesthetics with modern components and smooth transitions.
- **Multi-Tenant Ready**: Scalable architecture for diverse retail environments.

## 🛠️ Tech Stack

- **Framework**: Next.js (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS / Vanilla CSS
- **Database**: Prisma (ORM)
- **State Management**: React Context / Hooks

## 🏁 Getting Started

### Prerequisites

- Node.js (Latest LTS recommended)
- npm / yarn / pnpm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/zyaddhamedd/juton-pos.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up environment variables:
   Copy `.env.example` to `.env` and configure your database and other settings.

4. Run the development server:
   ```bash
   npm run dev
   ```

## 🏗️ Project Structure

- `src/app`: Next.js pages and layouts.
- `src/components`: Reusable UI components.
- `src/lib`: Utility functions and shared logic.
- `src/hooks`: Custom React hooks.
- `prisma/`: Database schema and migrations.
